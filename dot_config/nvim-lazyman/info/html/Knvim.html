Lazyman config: Knvim

Knvim Neovim Configuration Information

Targets Python, Bash, LaTeX, Markdown, and C/C++. See the Knvim Config Cheat Sheet

  • Install and initialize: lazyman -L Knvim
  • Configuration category: Language
  • Base configuration: Custom
  • Plugin manager: Lazy
  • Installation location: ~/.config/nvim-Knvim
